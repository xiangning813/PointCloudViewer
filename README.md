一个基于 winform 开发的点云数据可视化工具。

## 功能特点

- 支持加载和显示点云数据文件
- 3D 场景实时渲染和交互
- 点云数据基本操作（旋转、缩放、平移）
- 点云颜色和大小调节
- 视角控制和场景导航

## 系统要求

- Windows 操作系统
- .NET Framework 4.7.2 或更高版本
- 支持 DirectX 的显卡

## 安装说明

1. 下载最新发布版本
2. 解压文件到本地目录
3. 运行 PointCloudViewer.exe

## 使用说明

### 基本操作

- 鼠标左键：旋转视角
- 鼠标右键：平移场景
- 鼠标滚轮：缩放场景
- 空格键：重置视角

### 文件操作

- 支持导入 .pcd、.ply 等常见点云文件格式
- 文件拖拽导入功能
